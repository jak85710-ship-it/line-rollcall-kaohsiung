# 純前端點名表 · 設定教學（免伺服器）

不用 Node.js、不用 Render、不用 LINE 官方帳號、**不用 Google Apps Script**。  
只要 **GitHub Pages 免費網址** + **EmailJS 寄信**。

---

## 運作方式

```
手機開啟固定網址 → 密碼 12345 → 點名 → 送出點名表
        ↓
EmailJS（免費）→ 寄到 ben83127@gmail.com
```

---

## 第一步：取得固定免費網址（GitHub Pages）

1. 開啟 https://github.com/jak85710-ship-it/line-rollcall-kaohsiung/settings/pages
2. **Source** 選 `Deploy from a branch`
3. **Branch** 選 `main`，資料夾選 **`/docs`**
4. 儲存，等 1–2 分鐘

固定網址：

```
https://jak85710-ship-it.github.io/line-rollcall-kaohsiung/
```

把此連結貼到 **LINE 群組記事本**。

---

## 第二步：設定 EmailJS 寄信

詳細步驟見：**[EmailJS設定教學.md](./EmailJS設定教學.md)**

摘要：
1. 到 [emailjs.com](https://www.emailjs.com) 註冊
2. 連接 Gmail（ben83127@gmail.com）
3. 建立模板：`To: {{to_email}}`、`Subject: {{subject}}`、`HTML: {{{message_html}}}`
4. 把 Public Key、Service ID、Template ID 填入 `docs/index.html` 的 `CONFIG.EMAILJS`

---

## 第三步：測試

1. 開啟點名表網址，密碼 `12345`
2. 每位各選一項 ○ 狀態
3. 按 **送出點名表**
4. 確認 ben83127@gmail.com 收到 Email

---
