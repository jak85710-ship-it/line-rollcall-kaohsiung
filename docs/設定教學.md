## 常見問題

### 畫面跟以前不一樣？
執行 `npm run build-static` 會從 `public/admin/index.html` 重新產生，保證一致。

### 送出失敗？
第一次請到 ben83127@gmail.com 點 FormSubmit 啟用連結（含垃圾郵件匣）。

### 新增/編輯隊員會不見？
資料存在瀏覽器 localStorage，同一台裝置會保留。清除瀏覽器資料會回到內建名單。
