# 純前端點名表 · 設定教學（免伺服器）

不用 Node.js、不用 Render、不用 LINE 官方帳號。  
只要 **GitHub Pages 免費網址** + **Google Apps Script**（完全免費）。

---

## 運作方式

```
手機開啟固定網址 → 輸入密碼 12345 → 每位各選一項 ○ → 送出點名表
        ↓
Google Apps Script（Google 免費提供）
        ↓
  ① 寫入 Google 試算表
  ② Gmail 寄出點名日報（含缺席學生家長電話）
```

---

## 第一步：取得固定免費網址（GitHub Pages）

你的 repo 已有程式，只需開啟 GitHub Pages（**一次性，約 2 分鐘**）：

1. 開啟 https://github.com/jak85710-ship-it/line-rollcall-kaohsiung/settings/pages
2. **Source** 選 `Deploy from a branch`
3. **Branch** 選 `main`，資料夾選 **`/docs`**
4. 儲存，等 1–2 分鐘

完成後固定網址為：

```
https://jak85710-ship-it.github.io/line-rollcall-kaohsiung/
```

把此連結貼到 **LINE 群組記事本**，教練點開即可使用。

---

## 第二步：設定 Google 試算表 + Apps Script

### 1. 建立試算表

1. 前往 [Google Sheets](https://sheets.google.com) 新增空白試算表
2. 命名為「桌球隊點名紀錄」
3. 複製網址中的 **Spreadsheet ID**：
   `https://docs.google.com/spreadsheets/d/`**這段**`/edit`

### 2. 建立 Apps Script

1. 試算表 → **擴充功能** → **Apps Script**
2. 刪除預設內容，貼上 `docs/google-apps-script.gs` 的全部程式碼
3. 修改 `CONFIG`：

```javascript
const CONFIG = {
  SPREADSHEET_ID: '你的試算表ID',
  LOG_SHEET: '點名紀錄',
  EMAIL_TO: 'ben83127@gmail.com',
  API_SECRET: 'zhongshan2026',  // 須與 index.html 相同
  TEAM_NAME: '中山國小桌球隊',
};
```

4. 按 **儲存**（Ctrl+S）

### 3. 部署為網頁應用程式

1. 右上角 **部署** → **新增部署作業**
2. 類型：**網頁應用程式**
3. 設定：
   - 說明：點名 API
   - 執行身分：**我**
   - 存取權：**任何人**（必選，前端才能呼叫）
4. **部署** → 第一次需 **授權** Google 帳號（允許寄信、寫試算表）
5. 複製 **網頁應用程式 URL**，例如：
   ```
   https://script.google.com/macros/s/AKfycb.../exec
   ```

### 4. 貼到 index.html

編輯 `docs/index.html`，找到 `CONFIG`，把 `GAS_URL` 改成上面的部署網址：

```javascript
const CONFIG = {
  PASSWORD: '12345',
  API_SECRET: 'zhongshan2026',
  GAS_URL: 'https://script.google.com/macros/s/你的ID/exec',
};
```

改完後 push 到 GitHub，GitHub Pages 會自動更新。

## 常見問題

### 送出失敗 / CORS 錯誤
- 確認 GAS 部署為「**任何人**」可存取
- 修改 GAS 程式後需 **重新部署**（建立新版本）

### 收不到 Email
- 檢查 GAS 授權是否完成
- 看 Apps Script → **執行作業** 是否有錯誤

### 忘記密碼
- 預設 `12345`，可在 `index.html` 的 `CONFIG.PASSWORD` 修改
