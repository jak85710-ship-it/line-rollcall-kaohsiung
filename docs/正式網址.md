## 一鍵部署到 Render

1. 開啟（需登入 Render + 授權 GitHub）：

   **https://render.com/deploy?repo=https://github.com/jak85710-ship-it/line-rollcall-kaohsiung**

2. 服務名稱確認為 **`csh-tt-rollcall`**（網址不含您的 GitHub 帳號）

3. 在 **Environment** 設定 Email（**最重要**）：

   | 變數 | 值 |
   |------|-----|
   | `SMTP_PASS` | Gmail **應用程式密碼**（16 碼，不含空格） |

   其餘已預設：
   - `SMTP_USER` = `ben83127@gmail.com`
   - `EMAIL_TO` = `ben83127@gmail.com`

   **Gmail 應用程式密碼取得方式：**
   1. 登入 Google 帳號 → [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords)
   2. 需先開啟「兩步驟驗證」
   3. 建立應用程式密碼（名稱可填：點名系統）
   4. 複製 16 碼密碼，貼到 Render 的 `SMTP_PASS`
   5. 按 **Save Changes**，Render 會自動重新部署

4. 部署完成後登入點名表，按 **「測試寄信」** 確認收到信

5. 按 **Deploy**，等 2–3 分鐘

5. 測試點名表（**不要**開 `/health`）：

   **https://csh-tt-rollcall.onrender.com/**
