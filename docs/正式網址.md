## 一鍵部署到 Render

1. 開啟（需登入 Render + 授權 GitHub）：

   **https://render.com/deploy?repo=https://github.com/jak85710-ship-it/line-rollcall-kaohsiung**

2. 服務名稱確認為 **`csh-tt-rollcall`**（網址不含您的 GitHub 帳號）

3. 在 Environment 設定以下三項：

   | 變數 | 值 |
   |------|-----|
   | `SMTP_USER` | 您的 Gmail（例如 `yourname@gmail.com`） |
   | `SMTP_PASS` | 您的 Gmail 應用程式密碼 |
   | `EMAIL_TO` | 您要收信的 Email（可與 `SMTP_USER` 相同） |

4. 按 **Deploy**，等 2–3 分鐘

5. 測試點名表（**不要**開 `/health`）：

   **https://csh-tt-rollcall.onrender.com/**

## 關閉 GitHub Pages（避免舊網址外流）

1. 開啟：https://github.com/jak85710-ship-it/line-rollcall-kaohsiung/settings/pages
2. **Source** 改選 **None**（停用）
3. 儲存

之後外人就不會再看到 `jak85710-ship-it.github.io` 的網址。

若暫時未停用，舊 GitHub Pages 會自動跳轉到新網址。
